# ai-training-exercises
A repo for exercises in learning AI.

## How to Start an Exercise

You can use the **Scaffold Exercise** skill to generate the boilerplate for any exercise in this repository.

1. **Pick an exercise** from the table below.
2. **Open the folder** (e.g., `01-llm-eval-harness`).
3. **Ask the Agent**: "Find a dataset for this exercise" or "Get data for 01-llm-eval-harness".
   - The agent will use the `data` skill to find and download a suitable dataset.
4. **Ask the Agent**: "Scaffold this exercise" or "Scaffold the exercise in 01-llm-eval-harness".
   - The agent will use the `scaffold-exercise` skill to generate the code structure, dependencies, and instructions.

## AI Engineering Learning Roadmap (Prioritized)

Below is a **prioritized learning table** (most important first). Each row has:

* **Difficulty** (Beginner / Intermediate / Advanced / Expert)
* **Time estimate** (realistic “interview-home-task” scale)
* **Exercise (.md content)**: written like a take-home assignment (requirements, deliverables, bonus)

> Tip: Treat each row as a GitHub repo. The deliverables are what you’d send to a hiring manager.
