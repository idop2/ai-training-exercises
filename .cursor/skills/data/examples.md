# Data Examples

## Good (Interesting)
- **Sci-Fi Books**: Project Gutenberg texts of H.G. Wells or Jules Verne.
- **Movie Dialogues**: Cornell Movie-Dialogs Corpus.
- **Space News**: NASA APOD API data.
- **Crypto Prices**: Historical BTC/ETH prices for time-series.
- **Recipe Ingredients**: Recipe1M+ dataset for cooking.

## Bad (Boring)
- **Iris**: Too simple, overused.
- **Titanic**: Too simple, overused.
- **Generic Lorem Ipsum**: Not real data.
- **Synthetic**: Generated data without context.
